# OTBM2 Bundle

This directory holds OTBM2 files usable with the game server, client and editor.

## Layout

- `server/`: Game-server OTBM2 inputs
- `client/`: Client OTBM2 assets
- `editor/`: Editor OTBM2 assets

## Current Files

### `server/`
- `world.otbm2`, example world
- `tibia-world.otbm2.zip`, Tibia 7.6 world

### `client/`
- `data.otbm2`
- `sprites.otbm2`

### `editor/`
- `world.otbm2`
- `data.otbm2`, same as client/
- `sprites.otbm2`, same as client/

## Source Mapping

- `server/world.otbm2` <- `gameserver/data/world.otbm2`
- `client/data.otbm2` <- `client/dat/data.otbm2`
- `client/sprites.otbm2` <- `client/dat/sprites.otbm2`
- `editor/world.otbm2` <- `editor/data/world.otbm2`
- `editor/data.otbm2` <- `editor/data/data.otbm2`
- `editor/sprites.otbm2` <- `editor/data/sprites.otbm2`
